# Fotos reales para la web

Guarda aquí fotografías propias y autorizadas en formato WebP o JPG.

- 3 pares de antes/después de reparaciones.
- 2 fotos del mostrador o de las sucursales.
- 2 fotos del equipo técnico trabajando.

Después añade cada archivo a `galleryItems` en `src/data/siteData.js` con `src`,
`alt`, `title`, `text` y `category`. La galería permanece oculta mientras no haya
fotos para evitar mostrar placeholders como si fueran prueba social.
